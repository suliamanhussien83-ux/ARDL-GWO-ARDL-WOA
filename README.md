# ARDL-GWO-ARDL-WOA
R implementation of ARDL-OLS, ARDL-GWO, and ARDL-WOA models for public expenditure forecasting. The repository includes data preprocessing, model estimation, optimization algorithms, forecasting, and evaluation metrics.
# ARDL-GWO-ARDL-WOA

## Overview
This repository contains the R implementation of ARDL-OLS, ARDL-GWO, and ARDL-WOA models for forecasting public expenditures.

## Contents
- Data preprocessing
- ARDL-OLS estimation
- Grey Wolf Optimizer (GWO)
- Whale Optimization Algorithm (WOA)
- Forecasting
- Performance evaluation
- Diebold–Mariano test

## Software
- R (version 4.x)
- Required packages:
  - forecast
  - tseries
  - Metrics
  - ggplot2
  - dplyr
